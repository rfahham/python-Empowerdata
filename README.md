## Isolar o ambiente: 

```bash 
python3 -m venv ./venv && source venv/bin/activate
```

## Salvar os requirements em um arquivo

```bash 
pip3 freeze > requirements.txt
```

Instalando os requeirementes a partir do arquivo

```bash 
pip install -r requirements.txt
```

## Sair do ambiente 

```bash 
deactivate
```
